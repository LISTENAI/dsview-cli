<objective>
Prove the lowest-risk native integration strategy against the `DSView/` submodule and document the constraints so later plans can build on a real boundary instead of assumptions.
</objective>

<scope>
This plan validates compile/link strategy and documentation. It does not yet implement device discovery or capture execution.
</scope>
