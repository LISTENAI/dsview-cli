## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Compiled CLI binary -> user scripts | Help text and parse diagnostics are part of the automation contract once users script the new flags |
| `devices options` inspection -> `capture` flag usage | Token drift between discovery and execution would cause valid-looking commands to fail or mislead users |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-12-03-02 | T | `crates/dsview-cli/tests/device_options_cli.rs` | mitigate | Assert the exact token values and flag order shown by `devices options` so discovery and execution cannot drift |
